# O-RADS Pro (единый экран)

## Что реализовано

- Один скроллящийся экран с зависимыми блоками (без пошаговой навигации)
- Детерминированный локальный расчет `O-RADS` в `logic/oradsCalculator.ts`
- Офлайн-сохранение истории кейсов (`AsyncStorage`)
- Кнопка `Мнение AI (Beta)`:
  - при доступном API отправляет сразу;
  - при недоступности добавляет запрос в офлайн-очередь;
  - очередь периодически повторно отправляется автоматически.
- Экспорт заключения через копирование в буфер обмена.
- История кейсов с фильтрами (категории, 7 дней, high-risk, критичные) и экраном деталей.
- UX метрика TTFR (time to first result) + среднее локально.
- Статусы AI-очереди (`idle/pending/success/failed`) и retry/backoff.

## Запуск

```bash
npm install
npm run start
```

## Опциональная AI-конфигурация

AI работает в мок-режиме, если переменные не заданы.

В `app.config.js` или `app.json` (через `extra`) можно добавить:

- `openRouterUrl`
- `openRouterKey`

## Ограничения текущего инкремента

- Алгоритм реализован в формате практического локального rule-engine и покрывает заявленные ветви O-RADS.
- При необходимости можно расширить до полного дерева v2022 с точной матрицей признаков и дополнительными ветвями исключений.
