# Release QA Checklist (O-RADS + BI-RADS)

## O-RADS Pro: Core flow

- [ ] Один экран без step-навигации, скролл работает стабильно.
- [ ] Зависимые блоки корректно появляются/скрываются.
- [ ] Экстраовариальный вариант показывает предупреждение.
- [ ] Карточка результата обновляется мгновенно при смене признаков.
- [ ] Цвета категорий соответствуют риску (1-2/3/4/5).

## O-RADS Pro: Algorithm regression

- [ ] Физиологическое (пременопауза, <=3 см) -> O-RADS 1.
- [ ] Простая однокамерная киста (типичная) -> O-RADS 2.
- [ ] Простая киста >5 см у постменопаузы -> O-RADS 3.
- [ ] Солидный компонент + умеренный/выраженный кровоток -> >= O-RADS 3.
- [ ] Асцит без другой причины -> минимум O-RADS 4.
- [ ] Асцит + перитонеальные высыпания + подозрительный паттерн -> O-RADS 5.
- [ ] Размер >10 см в постменопаузе поднимает риск на ступень (до 4).

## O-RADS Pro: Offline + AI queue

- [ ] Без сети расчет O-RADS полностью работает локально.
- [ ] Кнопка "Мнение AI (Beta)" без сети ставит запрос в очередь.
- [ ] При появлении сети очередь отправляется автоматически.
- [ ] На экране видны queue-size, статус (idle/pending/success/failed), текст ошибки/retry.

## O-RADS Pro: History

- [ ] Кейс сохраняется в историю автоматически.
- [ ] Фильтры работают: категория, 7 дней, O-RADS 4-5, критичные.
- [ ] Сортировки работают: новые/риск.
- [ ] Экран деталей открывается из истории.
- [ ] "Использовать как шаблон" и "Дублировать как новый кейс" заполняют форму.

## BI-RADS regression (обязательно перед релизом)

- [ ] Базовый BI-RADS калькулятор открывается и не падает.
- [ ] Типичный benign cystic pattern -> низкая категория (2/3 в рамках ruleset).
- [ ] Подозрительные признаки (irregular/non-parallel/marked flow) -> повышенная категория.
- [ ] Decision path генерируется без пустых шагов.
- [ ] Версия BI-RADS ruleset отображается корректно.
- [ ] Изменения O-RADS не ломают BI-RADS роутер и сценарии.

## UX KPI

- [ ] TTFR (time to first result) отображается.
- [ ] Средний TTFR сохраняется локально и обновляется.
- [ ] Целевое TTFR < 5 сек на типовом клиническом сценарии.
